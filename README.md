Website quản lý quán cà phê, hỗ trợ quản lý sản phẩm, khách hàng và dữ liệu liên quan.

## 🚀 Features
- Hiển thị danh sách sản phẩm
- Thêm / sửa / xóa dữ liệu (CRUD)
- Quản lý thông tin cơ bản của hệ thống

## 🛠️ Technologies
- HTML, CSS, JavaScript
- C# (.NET)
- SQL Server

## 👨‍💻 My Role
- Tham gia phát triển giao diện người dùng
- Xử lý các chức năng CRUD
- Kết nối và làm việc với database
- Fix bug và cải thiện chức năng

## ▶️ How to run
1. Mở project bằng Visual Studio
2. Cấu hình database trong file appsettings.json
3. Chạy project
